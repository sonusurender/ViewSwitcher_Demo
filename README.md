# ViewSwitcher
ViewSwitcher is a sub-class of ViewAnimator, switches between two views, and has a factory from which these views are created. You can either use the factory to create the views, or add them yourself.

A ViewSwitcher can only have two child views, of which only one is shown at a time. If you have more than two child views in ViewSwitch, **java.lang.IllegalStateException** of **‚ÄúCan‚Äôt add more than 2 views to a ViewSwitcher‚Äù** will happen.
